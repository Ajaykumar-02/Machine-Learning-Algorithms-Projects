# 🤖 Machine Learning Algorithms: Implementation & Notes

A comprehensive repository containing my personal coding notes, mathematical foundations, and hands-on implementations of various Machine Learning algorithms.

## 📌 Project Overview
The goal of this repository is to document my ML journey by implementing core algorithms from scratch (using NumPy/Pandas) as well as using industry-standard libraries like **Scikit-Learn**, **XGBoost**, and **LightGBM**.

---

## 🛠️ Tech Stack
* **Language:** Python
* **Data Analysis:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **ML Libraries:** Scikit-Learn, Scipy
* **Environment:** Jupyter Notebooks / VS Code

## 📝 Core Concepts Covered
Beyond just code, this repo includes notes on:
* **Feature Engineering:** Scaling, Encoding, and Handling Missing Data.
* **Model Evaluation:** Confusion Matrix, Precision-Recall, F1-Score, ROC-AUC, and R-Squared.

## 🚀 How to Navigate
1.  **Folders:** Each folder is named after a specific algorithm.
2.  **Notebooks:** Inside each folder, you'll find a `.ipynb` file containing the explanation, math, and code.
3.  **Data:** Small datasets used for practice are located in the `data/` directory.
